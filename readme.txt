== Description ==

Professional solution for Dynamic Pricing and Discounts.

**[Plugin Demo](http://azexo.com/automation/)**

This is add-on for **[Marketing Automation by AZEXO](https://wordpress.org/plugins/marketing-automation-by-azexo/)**.


= Main Features =

* Product price adjustment (percentage discount, fixed discount, fixed price)
* Bulk pricing (percentage discount, fixed discount, fixed price) **[in PRO version](https://codecanyon.net/item/woocommerce-dynamic-pricing-and-discounts/21655238)**
* Tiered pricing (percentage discount, fixed discount, fixed price) **[in PRO version](https://codecanyon.net/item/woocommerce-dynamic-pricing-and-discounts/21655238)**
* "Buy x and get x" pricing (percentage discount, fixed discount, fixed price) **[in PRO version](https://codecanyon.net/item/woocommerce-dynamic-pricing-and-discounts/21655238)**
* Cart discount (percentage discount, fixed discount)
* Show dynamic text inside product template with any position and visibility rules
* Wide range of cart content conditions
* Wide range of customer purchases history conditions
* Wide range of customer conditions
* Wide range of products filtering conditions
* Possibility for AND/OR/NOT conditions combinations (logical expressions)
* Exclusivity settings for rules overlapping case
* Overlapped rule selection methods for bigger/smaller price/discount
* Supported all features of **[Marketing Automation by AZEXO](https://wordpress.org/plugins/marketing-automation-by-azexo/)**


== Installation ==

1. Upload the `azh_woo_discounts` folder to your plugins directory (e.g. `/wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Install Marketing Automation by AZEXO plugin
4. Add new rule 
5. Select product pricing or cart discount "Action"
6. Set event which required by "Action"
7. Set conditions you need
